   NAME
          conda - conda develop
   
   DESCRIPTION
          usage:  conda-develop [-h] [-npf] [-n ENVIRONMENT | -p PATH] PATH [PATH
          ...]
   
          Install a Python package in 'development mode'.
   
          This works by creating a conda.pth file in site-packages.
   
   OPTIONS
      positional arguments:
          PATH   Path to the source directory.
   
      optional arguments:
          -h, --help
                 Show this help message and exit.
   
          -npf, --no-pth-file
                 Relink compiled extension dependencies against  libraries  found
                 in current conda env. Do not add source to conda.pth.
   
          -n ENVIRONMENT, --name ENVIRONMENT
                 Name of environment (in envs dirs).
   
          -p PATH, --prefix PATH
                 Full  path to environment prefix (default: envs dirs/build-envi‚Äê
