   NAME
          conda - conda build
   
   DESCRIPTION
          usage: conda-build [-h] [-c] [--no-binstar-upload] [--output] [-s] [-t]
   
                 [--no-test]  [-b]  [-p]  [-V] [-q] [--python PYTHON_VER] [--perl
                 PERL_VER] [--numpy NUMPY_VER] RECIPE_PATH [RECIPE_PATH ...]
   
          tool for building conda packages
   
      positional arguments:
          RECIPE_PATH
                 path to recipe directory
   
      optional arguments:
          -h, --help
                 show this help message and exit
   
          -c, --check
                 only check (validate) the recipe
   
          --no-binstar-upload
                 do not ask to upload the package to binstar
   
          --output
                 output the conda package filename which would have been  created
                 and exit
   
          -s, --source
                 only obtain the source (but don't build)
   
          -t, --test
                 test package (assumes package is already build)
   
          --no-test
                 do not test the package
   
          -b, --build-only
                 only  run  the  build,  without  any post processing or testing.
                 Implies --no-test and --no-binstar-upload
   
          -p, --post
                 run  the  post-build  logic.  Implies  --no-test  and   --nobin‚Äê

          -V, --version

          -q, --quiet
                 do not display progress bar
   
          --python PYTHON_VER
                 Set the Python version used by conda build
   
          --perl PERL_VER
                 Set the Perl version used by conda build
   
          --numpy NUMPY_VER
                 Set the NumPy version used by conda build
