   NAME
          conda - conda inspect objects
   
   DESCRIPTION
          usage: conda-inspect objects [-h] [--untracked]
   
          [--groupby {filetype,filename,rpath}] [--all]
                 [-n ENVIRONMENT | -p PATH] [packages [packages ...]]
   
          Investigate  binary  object  files  in  a package (only works in OS X).
          Aggregates the output of otool on all the  binary  object  files  in  a
          package.
   
      positional arguments:
          packages
                 conda packages to inspect
   
      optional arguments:
          -h, --help
                 show this help message and exit
   
          --untracked
                 Inspect the untracked files in the environment. Useful when used
                 in conjunction with conda build --buildonly.
   
          --groupby {filetype,filename,rpath}
                 Attribute to group by (default: filename)
   
          --all  Generate a report for all packages in the environment.
   
          -n ENVIRONMENT, --name ENVIRONMENT
                 Name of environment (in envs dirs).
   
          -p PATH, --prefix PATH
                 Full path to environment prefix (default: envs  dirs/build-envi‚Äê
                 ronment).
