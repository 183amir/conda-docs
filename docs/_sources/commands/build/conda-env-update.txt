   DESCRIPTION
          usage: conda-env update [-h] [-n NAME] [-f FILE] [-q QUIET] [--json]

      optional arguments:
          -h, --help
                 show this help message and exit
   
          -n NAME, --name NAME
                 name of environment (in envs dirs)
   
          -f FILE, --file FILE
                 environment definition (default: environment.yml)
   
          -q QUIET, --quiet QUIET
