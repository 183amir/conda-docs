   NAME
          conda - conda bundle
   
   DESCRIPTION
          usage: conda bundle [-h] [-c | -x PATH | --metadump PATH]
   
          [-n ENVIRONMENT | -p PATH] [-q] [--bundle-name NAME]
                 [--data-path PATH] [--extra-meta PATH] [--no-env] [--json]
   
          Create or extract a "bundle package" (EXPERIMENTAL)
   
      optional arguments:
          -h, --help
                 show this help message and exit
   
          -c, --create
                 create bundle
   
          -x PATH, --extract PATH
                 extact bundle located at PATH
   
          --metadump PATH
                 dump metadata of bundle at PATH
   
          -n ENVIRONMENT, --name ENVIRONMENT
                 name of environment (in envs dirs)
   
          -p PATH, --prefix PATH
                 full  path to environment prefix (default: envs dirs/build-envi‐
                 ronment)
   
          -q, --quiet
                 do not display progress bar
   
          --bundle-name NAME
                 name of bundle
   
          --data-path PATH
                 path to data to be included in bundle
   
          --extra-meta PATH
                 path to json file with additional meta-data
   
          --no-env
                 no environment
   
          --json report all output as json. Suitable for using conda programmati‐
                 cally.
